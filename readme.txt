== Description ==



With this wp antispam plugin you can stop spammers commenting on your blog with poor comments looking for links.


Features:
Simple to use and easy to install, this will add a layer of antispam protection to your blog.



== Installation ==


1. Unzip files.

2. Upload 'wp-antispam.zip' to the '/wp-content/plugins/' directory

3. Activate the plugin through the 'Plugins' menu in WordPress

4. Open "wp-config.php" in WordPress root directory.

5. Add "define('EMPTY_TRASH_DAYS', <Days> );". 

Specify after how many days basket will be cleaned automatically. 
For example, "define('EMPTY_TRASH_DAYS', 1 );" or "define('EMPTY_TRASH_DAYS', 5 );".
